元通商圈

商圈概况介绍

元通商圈位于南京市建邺区，共143万平方米，属于新兴高端商圈，是高能级综合性商业功能区。

元通商圈地处河西奥体新城区域，位于长江东岸，地理位置优越，商业势能强劲。在南京市最新的商业规划中属于二级商圈，仅次于新街口商圈。2024年1月，元通商圈南京IFC正式营业，后续还将迎来河西德基广场，高端项目的接连亮相也将直接拉升元通商圈的商业能级。

南京IFC商场

由新鸿基地产倾力打造的南京国金中心坐落于南京市河西中央商业区核心，位处南京地铁元通站上盖，无缝连接地铁2号线和10号线。 项目楼面面积约30万平方米，包括高端购物中心、两幢优质办公楼及南京河西安达仕酒店。

结合上海国金中心商场的成功经验，南京国金中心商场(南京ifc商场）将以高端时尚作定位，云集国际顶级品牌，部分更是首次登陆南京，提供一站式崭新的购物、娱乐及消闲体验。 商场位于三座大楼的基座部分，总楼面面积约10万平方米，共分九层，商户约168间。

地址：南京市建邺区江东中路345号
营业时间：10:00-22:00（周末无延长）
定位：重奢商业体
特色：顶奢品牌旗舰矩阵、高规格门店设计

华采天地

华采天地，位于河西中部，江东中路258号，是由华新（南京）置业开发有限公司开发建设的大型商场。

作为台资企业华新丽华集团在南京的首个大型商业项目，承载着集团董事长焦佑伦"回报家乡"的战略愿景。项目总投资额未公开，但楼顶设备优化工程单项投资达3000万元 。2018年3月招商发布会公布"ONE MALL模式"运营理念，由原南京景枫KINGMO操盘手吴正梅担任总经理。

地址：南京市建邺区江东中路258号
营业时间：10:00-22:00（周末无延长）
定位：大型商业综合体
特色：健康生活 + 亲子同乐 + 美学体验
